Class TableRow

public class TableRow

Classe TableRow<br /> Implements methods of manipulation of a row of the one html table.
It makes possible to get a cell object or set cell object value, to get style and the properties of the group of the cell, to change the style and the properties of the group of the cell, to concatenate content to the value of the cell and to clean properties of the group, of a specific type, of the cell.
