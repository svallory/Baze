isPrimaryKey

private static boolean isPrimaryKey(string $tableName, string $fieldName)

Function isPrimaryKey() This function checks if a field is primary key or not.
Through the parameter $tableName, that contains the table name of the field, a store procedure is called and executed through sending the table name. If the third property [COLUMN_NAME] possess the name of the field, means that this is primary key.

Parameters

$tableName - - name of specified table
$fieldName - - name of field containded in the table
