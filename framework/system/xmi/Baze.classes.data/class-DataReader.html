fetchRow

public DataRow fetchRow()

Function fetchRow<br> This function has as objective to create and to return a DataRow object.
The function gets an Array contends the values of the current line of the result using the command fetchRow of driver. For each value of the Array, a DataField object is created and kept in one another Array.
In the end, is created a DataRow object sending Array DataField the object and returned this object.
