Class SQLParser

public class SQLParser

Classe SQLParser<br /> This class receives a command SQL and transforms the properties of the fields that will be generated by the query (not NonQuery) in objects. Each identified field is generated an object with the properties of this field and kept in a Collection object. The class does not make total validation of query. The same one must be valid and linear (string).
