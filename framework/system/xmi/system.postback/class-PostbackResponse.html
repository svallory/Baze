addJSFunction

public void addJSFunction(string $func, mixed $parms)

Adds a javascript function call to be executed on the client
The parameters specified on $params must be EXACTLY as you would write them on javascript.
For instance, if you want to pass a string you have to put the quotes as part of the value of the parameter you are passing here.

Examples:
->addJSFunction('alert', array('"a message!"')); # alert("a message!"); ->addJSFunction('myFunc', array('1', 'aVar', '"a string"')); # myFunc(1, aVar, "a string");

Parameters

$func - The function name
$parms - The parameters, exactly as they should be writen on the javascript
